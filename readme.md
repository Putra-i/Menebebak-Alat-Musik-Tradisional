# Mini Game "Menebebak Alat Musik Tradisional"
Sebuah permainan interaktif berbasis web untuk menebak alat musik tradisional Indonesia, dibangun dengan React, Three.js, dan teknologi web modern.

## Tentang
Tebak Alat Musik Tradisional adalah aplikasi edukatif dan menghibur yang membantu pengguna mempelajari alat musik tradisional Indonesia melalui pengalaman 3D yang interaktif.

## Fitur
* Visualisasi 3D Interaktif - Dibangun dengan Three.js dan React Three Fiber
* Animasi Halus - Didukung oleh Framer Motion
* Manajemen State - Pengelolaan state yang efisien dengan Zustand
* Desain Responsif - Berfungsi dengan baik di berbagai ukuran layar
* Konten Edukatif - Belajar tentang warisan budaya Indonesia

## Prasyarat
Sebelum memulai, pastikan Anda telah menginstal:

* [Node.js](https://nodejs.org/en/download) (versi 12.7.0 atau lebih tinggi)
* [Git Bash](https://git-scm.com/download/win) - Terminal untuk menjalankan perintah (sudah termasuk saat install Git)
* [npm](https://www.npmjs.com/get-npm) (sudah termasuk saat install Node.js)

## Cara Menggunakan
Instalasi

Jalankan development server melalui Git Bash:

```bash
npm run dev
```
Aplikasi akan tersedia di http://localhost:5173 (atau port lain jika 5173 sedang digunakan).

## Cara Bermain

1. Buka aplikasi di browser web Anda
2. Lihat representasi 3D dari alat musik tradisional
3. Tebak alat musik apa yang sedang ditampilkan
4. Pelajari fakta menarik tentang alat musik tradisional Indonesia
